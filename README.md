## 📋 Overview

A modern, responsive portfolio website built to showcase my work as an AI/ML Engineer and Data Analyst. This portfolio highlights my projects, skills, certifications, and achievements in a clean, professional design optimized for recruiters and potential collaborators.

## ✨ Features

- **🎨 Modern Design** - Clean, professional blue theme with grunge texture background
- **📱 Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **⚡ Fast Performance** - Lazy loading images and smooth animations
- **🔍 SEO Optimized** - Meta tags and structured content for better visibility
- **🎯 Recruiter-Focused** - Highlights skills, projects, and achievements prominently
- **🖱️ Interactive Elements** - Hover effects and smooth scrolling navigation

## 🛠️ Tech Stack

| Category | Technologies |
|----------|-------------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla) |
| **Styling** | CSS Grid, Flexbox, Custom Animations |
| **Fonts** | Custom Web Fonts (YADK4LSvvyE-0, YAFdJt8dAY0-0) |
| **Design** | Responsive Design, CSS Variables, Backdrop Filters |
| **Performance** | Intersection Observer API, Lazy Loading |

## 🔄 Performance Optimizations

- **Lazy Loading** - Images load only when in viewport
- **Intersection Observer** - Sections animate on scroll
- **CSS Hardware Acceleration** - Smooth transitions and transforms
- **Minimal JavaScript** - Lightweight and fast execution
- **Optimized Fonts** - WOFF2 format for faster loading
